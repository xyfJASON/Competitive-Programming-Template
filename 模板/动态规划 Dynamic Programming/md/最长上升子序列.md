<h2 style="text-align: center"> Longest Increasing Subsequence </h2>



**Code** ( $O(n\lg n)$ )：

```cpp
int dp[N];
int LIS(){
	memset(dp, 0, sizeof dp); // dp[i]: a substring whose length is i ends at dp[i]
	int len = 0;
	for(int i = 1; i <= n; i++){
		if(b[i].y >= dp[len])
			dp[++len] = b[i].y;
		else{
			int p = upper_bound(dp+1, dp+len+1, b[i].y) - dp;
			dp[p] = b[i].y;
		}
	}
	return len;
}
```
